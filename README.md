# SCP Foundation Archive

Projeto estatico em HTML e CSS pronto para publicacao no GitHub Pages.

## Arquivos

- `index.html`
- `styles.css`

## Como publicar

1. Crie um repositorio publico chamado `projeto-scp-foundation`.
2. Envie os arquivos deste projeto para esse repositorio.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch `main` e a pasta `/ (root)`.
6. Salve e aguarde a publicacao.
